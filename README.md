# simple-java-web-test
Simple Java Web Test

I've used the **Glassfish file realm** to store the users:

| user       | roles                  |
| ---------- | ---------------------- |
| user_one   | PAGE_1                 |
| user_two   | PAGE_2                 |
| user_three | PAGE_3                 |
| user_full  | PAGE_1, PAGE_2, PAGE_3 |
